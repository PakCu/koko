PENTADBIR SISTEM
1st Admin masukkan maklumat Kelas
2nd	Admin masukkan maklumat Kokurikulum - Senarai Badan Uniform, Kelab/Persatuan dan Sukan/Permainan. 
                                          Termasuk jawatan, penglibatan, pencapaian 
3rd	Admin masukkan maklumat guru
4th	Admin masukkan maklumat pelajar
5th	Admin masukkan maklumat Sesi Persekolahan 
						  	- Admin masukkan maklumat tahun
						  	- Admin assignkan kelas dan guru untuk tahun berkenaan, berdasarkan database kelas dan guru yg ada tadi
						  	- Admin assignkan pelajar utk kelas-kelas tadi, berdasarkan database pelajar tadi
						  	- Jadi main key kat sini adalah sesi persekolahan ni. Semua maklumat uniform/kelab/sukan yg
						      guru akan masukkan nanti adalah untuk sesi persekolahan yg terakhir dimasukkan oleh admin.
						      Contohnya, jika admin masukkan tahun ialah 2011, maka semua data daftar/markah koko 
						      yg akan dimasukkan oleh guru adalah untuk tahun berkenaan

PENGGUNA
1st	Login
2nd	Klik button Login - Perhatikan kat atas menubar (sebelah tarikh) akan dipaparkan nama guru yg login
3rd	Klik menu Daftar Kokurikulum - Paparan senarai pelajar yg telah/belum dimasukkan maklumat kokurikulum
4th	Klik mana2 link Kemaskini - Paparan ini utk mendaftarkan/mengemasakini maklumat kokurikulum pelajar
5th	Klik button Hantar - Maklumat kokurikulum pelajar di daftarkan dan dibawa ke halaman senarai pelajar balik
6th	Klik menu Markah Kokurikulum - Paparan senarai pelajar. Bagi pelajar yg telah dimasukkan markah koko akan dipaparkan
7th	Klik link Kemaskini - Paparan untuk kemasukan markah. Utk field Markah Total dan Gred, sistem akan auto calculate/generate
8th	Klik button Hantar - Maklumat markah dikemaskini
